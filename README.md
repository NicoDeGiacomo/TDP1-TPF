## Build Steps
Manually execute the following.
```shell
$ rm -r build
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./TESTS
$ ./SERVER
$ ./CLIENT
```

Or let our bash script do it four you!
```shell
$ sh build.sh
```
